# AG Grid (Salt Themed)

The Salt AG Grid component provides a powerful data grid solution with the look and feel of the Salt Design System. It includes features like search, column filtering, and row grouping.

## Key Features

*   Styled with the Salt theme.
*   Search functionality across all columns.
*   Enable/disable column filters.
*   Toggle row grouping by a specific column (e.g., 'Make').
